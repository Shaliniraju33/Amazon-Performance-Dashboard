# Amazon-Performance-Dashboard
Overview

This project is an interactive Amazon Performance Dashboard built using Power BI to analyze and visualize sales and business performance. 
It provides insights into key metrics like total sales, profit, orders, and customer trends.
Features
Sales, Profit and Order Analysis
Category and Product Performance
Customer and Region-wise Insights
Monthly and Yearly Trends
Interactive Filters and Slicers
Tools Used
Power BI
Power Query
Objective

The objective of this project is to present Amazon sales data in a clear and interactive way, helping to identify trends, track performance, and support data-driven decision-making.
